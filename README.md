Library Book Tracker (Python Project)

A simple command-line based Library Management System built using Python.
This project allows users to manage books efficiently by adding, viewing, searching, borrowing, returning, and deleting books.

Features Add new books View all books Search books by title Borrow books Return books Delete books Data stored in JSON file (persistent storage)

Technologies Used

Python
JSON (for data storage)
File Handling
